# cicd-sensor documentation

The rendered version of these docs lives at **<https://cicd-sensor.github.io/>**.

This directory holds the markdown source. Editing any file here triggers
a rebuild of the published site through the
[cicd-sensor.github.io](https://github.com/cicd-sensor/cicd-sensor.github.io)
repository, which owns the mdbook configuration, theme, and deploy
pipeline.

## Local preview

Clone `cicd-sensor` and `cicd-sensor.github.io` as siblings, then run
`./serve.sh` in the docs repo. See that repo's README for details.
